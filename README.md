# IT Asset Report System

A full CRUD web application developed for managing IT asset reports such as computers, notebooks, printers, and other devices.

##  Features
- Create, Read, Update, Delete (CRUD)
- Report submission form with validation
- Responsive UI design
- Modern CSS styling
- JavaScript input validation
- PHP backend logic
- MySQL database integration

##  Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

##  Project Structure
- addReport.html – Report submission form
- listReport.php – Display report list
- editReport.php – Edit report
- delReport.php – Delete report
- postReport.php – Insert data into database
- postEditReport.php – Update database record
- style.css – Main styling
- T.css / T.html – Animated toggle UI component
